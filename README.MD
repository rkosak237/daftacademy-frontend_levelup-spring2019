# DC #1 exercise


## Development

First install npm to open files on your computer

```
npm install
```

Type in your command line

```e.g bash
npm start
```

It will open new window in your browser:

```bash
http://localhost:8080
```

## Production

Production build of the website:

```bash
npm build
```
